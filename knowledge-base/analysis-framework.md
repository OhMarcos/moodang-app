| 필드 | 필수 | 형식 | 용도 |

| 출생 시간 | 필수 | HH:MM (24h) | 시주, Lagna, Nakshatra 정밀 계산 |
| 성별 | 필수 | 남/여 | 대운 순행/역행 결정 |
| 출생지 | 필수 | 도시 (위도/경도) | Vedic Lagna, 사주 절입일, 시차 보정 |

| 현재 질문/고민 | 권장 | 텍스트 | I Ching 주역 점괘 질문 설정 |

| 필드 | 필수 | 형식 | 용도 |
|------|------|------|------|

| 소통 빈도 | 필수 | 매일/주간/월간 | 에너지 영향력 가중치 |

## Lens 1: 사주팔자 — Enhanced Analysis

### A. Subject 원국 (기존)
- 사주 원국 도출 (년월일시 4주)
- 일간 분석, 오행 분포, 신강/신약
- 용신/희신/기신, 십신, 격국
- 합충형해, 대운/세운

### B. 위치 보정

#### 출생지 적용
- **절입일 보정**: 출생지 경도 기준으로 태양 절기 통과 시각 계산
- **시주 보정**: 출생지 진태양시(True Solar Time) 기반으로 시주 결정
- **보정 공식**: `진태양시 = 현지시간 + (표준경도 - 출생지경도) × 4분 + 균시차`

#### 거주지 적용 — 풍수 오행 매칭
| 지역 특성 | 오행 | 사주 적용 |
|-----------|------|-----------|
| 바다/강/호수 근처 | 水 | 수 부족 사주에 보완 |
| 산/고지대 | 土 | 토 부족 사주에 보완 |
| 숲/공원 풍부 | 木 | 목 부족 사주에 보완 |
| 도시/번화가/사막 | 火 | 화 부족 사주에 보완 |
| 광산/공업지대/금속건물 | 金 | 금 부족 사주에 보완 |

### C. 관계 궁합 분석

각 관계인에 대해:
- **일간 관계 분석**: 상생/상극/비겁 관계 파악
- **오행 보완 분석**: Subject의 부족한 오행을 관계인이 보유하는지 확인
- **합충 관계**: 천간합, 지지합, 충, 형, 해 분석

### D. 오행 보완 맵

Subject + 모든 관계인의 오행을 종합한 "관계 에너지 생태계"

## Lens 2: Vedic Dasha — Enhanced Analysis

### A. Birth Chart (기존)
- Sidereal 보정 (Lahiri Ayanamsha)
- Moon Nakshatra 기반 Dasha 계산
- Mahadasha/Antardasha 시퀀스

### B. Lagna (Ascendant) 계산 — 출생지 필수
- 출생 시각 + 출생지 위도/경도로 상승 별자리 계산
- Lagna에 따라 행성의 하우스 배치, 길흉 판단, Dasha 해석 방향 변경

### C. Relocational Analysis — 거주지
- Astro*Carto*Graphy 원리에 따른 행성 라인 분석
- 거주지 행성 에너지 강화/약화 판단

### D. Synastry — 관계 분석
- Nakshatra 호환성 (Ashtakoot 매칭)
- Dasha 동기화 패턴 분석
- Transit 교차점 (관계의 위기/기회 타이밍)

---

## Lens 3: 주역 (I Ching) — 실시간 의사결정 오라클

### A. 괘 도출 (Hexagram Casting)
- **매화역수 (Plum Blossom Method)**: 시간/숫자 기반 자동 도출

### B. 변효 해석 (Moving Lines)
| 요소 | 해석 |
|------|------|
| **본괘 (Primary Hexagram)** | 현재 상황의 본질 |
| **변효 (Moving Lines)** | 변화의 핵심 포인트, 가장 중요한 조언 |
| **지괘 (Transformed Hexagram)** | 상황이 전개될 방향 |

### C. 괘상 종합 — 사주 오행과의 교차 분석
- 도출된 괘의 오행이 Subject의 용신/기신과 어떤 관계인지 확인
- 괘의 오행이 용신과 같으면 → 긍정적 방향
- 괘의 오행이 기신과 같으면 → 신중한 접근

### D. 실천 지침 (Action Guidance)
주역은 다른 2개 렌즈가 제시하는 장기적 흐름 위에 "지금 이 순간의 지혜"를 더함

## Analysis Rules

### 데이터 품질 등급

| 등급 | 조건 | 분석 범위 |
|------|------|-----------|
| **A급** | 생년월일시 + 출생지 + 거주지 + 현재 질문 + 관계인 3명+ | 전체 3렌즈 분석 가능 |
| **B급** | 생년월일시 + 출생지 | Vedic Lagna 정밀, 주역 가능, 관계 제한 |
| **C급** | 생년월일시만 | 사주 + 주역 + Vedic 제한 |
| **D급** | 생년월일만 (시간 없음) | 시주 제외, 주역 가능, 사주/Vedic 정밀도 낮음 |

### 주의 사항

1. **관계인 분석은 Subject 중심**: Subject에게 미치는 에너지 영향만 분석
2. **출생 시간 부재 시**: 일간과 오행 분포만으로 궁합 분석. Vedic은 Moon sign 추정으로 제한
3. **확증편향 경계**: 균형 잡힌 해석을 위해 반드시 Devil's Advocate 섹션 포함
