## Input Schema

| 필드 | 필수 | 형식 | 용도 |
|------|------|------|------|

| 출생 시간 | 필수 | HH:MM (24h) | 시주, Lagna, Nakshatra 정밀 계산 |

| 현재 거주지 | 권장 | 도시 (위도/경도) | Relocational chart, 풍수 오행 매칭 |

| 현재 질문/고민 | 권장 | 텍스트 | I Ching 주역 점괘 질문 설정 |

| 소통 빈도 | 필수 | 매일/주간/월간 | 에너지 영향력 가중치 |

## Lens 1: 사주팔자 — Enhanced Analysis

### A. Subject 원국 (기존)
- 사주 원국 도출 (년월일시 4주)
- 일간 분석, 오행 분포, 신강/신약
- 용신/희신/기신, 십신, 격국
- 합충형해, 대운/세운

### B. 위치 보정 (NEW)

#### 출생지 적용
- **절입일 보정**: 출생지 경도 기준으로 태양 절기 통과 시각 계산. 절기 경계 ±1일 출생자는 월주가 달라질 수 있음
- **시주 보정**: UTC 기준이 아닌 출생지 진태양시(True Solar Time) 기반으로 시주 결정. 서울(E127°) vs 부산(E129°) = ~8분 차이
- **보정 공식**: `진태양시 = 현지시간 + (표준경도 - 출생지경도) × 4분 + 균시차`

#### 거주지 적용 — 풍수 오행 매칭
| 지역 특성 | 오행 | 사주 적용 |
|-----------|------|-----------|
| 바다/강/호수 근처 | 水 | 수 부족 사주에 보완 |
| 산/고지대 | 土 | 토 부족 사주에 보완 |
| 숲/공원 풍부 | 木 | 목 부족 사주에 보완 |
| 도시/번화가/사막 | 火 | 화 부족 사주에 보완 |
| 광산/공업지대/금속건물 | 金 | 금 부족 사주에 보완 |

- **판단 기준**: 용신/희신 오행과 거주지 오행이 일치하면 길, 기신 오행과 일치하면 흉
- **예시**: 金 과잉 + 水 부족 사주 → 바다/강 근처(水) 거주 = 설기(洩氣)로 균형 회복

### C. 관계 궁합 분석 (NEW)

각 관계인에 대해:

#### C-1. 일간 관계 분석
| 관계인 일간 vs Subject 일간 | 관계 유형 | 에너지 영향 |
|---------------------------|-----------|-------------|
| 상생 (내가 생함) | 식상/재성 관계 | 에너지 소모되지만 생산적 |
| 상생 (나를 생함) | 인성 관계 | 에너지 보충, 보호 |
| 상극 (내가 극함) | 재성/관성 관계 | 긴장이 있지만 제어 가능 |
| 상극 (나를 극함) | 관성/극신 관계 | 압박, 도전 (좋을 수도 나쁠 수도) |
| 동일/비겁 | 비견/겁재 관계 | 경쟁 또는 동지 |

#### C-2. 오행 보완 분석
- Subject의 부족한 오행을 관계인이 보유하는지 확인
- **보완도 점수**: 관계인의 사주에서 Subject에 부족한 오행이 얼마나 강한지 측정
- **위험도 점수**: 관계인의 사주가 Subject의 기신 오행을 강화하는지 확인

#### C-3. 합충 관계
- 두 사주 간 천간합, 지지합, 충, 형, 해 분석
- **천간합**: 甲己합, 乙庚합, 丙辛합, 丁壬합, 戊癸합 → 자연스러운 끌림
- **지지충**: 에너지 충돌 → 갈등 요소이나 자극/성장 가능
- **지지합**: 삼합/육합 → 안정적 조화 에너지

### D. 오행 보완 맵 (NEW)

Subject + 모든 관계인의 오행을 종합한 "관계 에너지 생태계":

```
Subject의 오행 분포:  金■■■■■ 土■■■■ 火■■ 木■ 水·

관계인 A (배우자):    水■■■■ 木■■■ 金■■ 火■ 土■   → 水木 보완 ✓
관계인 B (파트너):    火■■■■ 木■■■ 土■■ 水■ 金■   → 火木 보완 ✓
관계인 C (친구):      金■■■ 土■■■ 水■■ 火■ 木■    → 金土 강화 ✗ (기신)
─────────────────────────────────────────────────
종합 에너지 환경:     균형도 = ?/10
                      보완 갭: ?
                      주의 관계: ?
```

## Lens 2: Vedic Dasha — Enhanced Analysis

### A. Birth Chart (기존)
- Sidereal 보정 (Lahiri Ayanamsha)
- Moon Nakshatra 기반 Dasha 계산
- Mahadasha/Antardasha 시퀀스
- Transit 분석

### B. Lagna (Ascendant) 계산 (NEW — 출생지 필수)

- **Lagna**: 출생 시각 + 출생지 위도/경도로 계산되는 상승 별자리
- 같은 시간이라도 서울(N37.5°, E127°) vs 뉴욕(N40.7°, W74°)에서 Lagna가 완전히 다름
- Lagna가 바뀌면:
  - **행성의 하우스 배치** 전체가 변경
  - **각 행성의 길흉** 판단이 달라짐
  - **Dasha 기간의 해석** 방향이 바뀜

#### 보정 절차
1. 출생 시각을 UTC로 변환
2. Local Sidereal Time (LST) 계산: `LST = GST + (출생지 경도 / 15)`
3. Lagna 도출: LST를 기준으로 상승 Rashi 결정
4. 전체 하우스 시스템 재배치

### C. Relocational Analysis (NEW — 거주지)

- **Astro*Carto*Graphy (ACG) 원리**: 출생 차트의 행성 라인이 지구 표면에 투사됨
- 특정 도시에서는 특정 행성의 에너지가 강화됨:
  - Jupiter 라인 근처 = 확장, 행운, 교육 기회
  - Saturn 라인 근처 = 제약, 책임, 장기 구축
  - Venus 라인 근처 = 관계, 예술, 물질적 풍요
  - Mars 라인 근처 = 에너지, 갈등, 경쟁

#### 판단 기준
- 거주지가 Subject의 길조 행성 라인 위인지 확인
- Dasha에서 활성화된 행성과 거주지 라인의 일치 여부
- 이사 추천: 현재 Dasha 행성의 길조 라인이 지나는 도시

### D. Synastry — 관계 분석 (NEW)

각 관계인에 대해:

#### D-1. Nakshatra 호환성
- Subject와 관계인의 Moon Nakshatra 쌍 분석
- 전통적 **Ashtakoot 매칭** (36점 만점 중):
  | 항목 | 만점 | 내용 |
  |------|------|------|
  | Varna | 1 | 정신적 호환 |
  | Vashya | 2 | 영향력/지배 패턴 |
  | Tara | 3 | 행운/건강 호환 |
  | Yoni | 4 | 물리적/성적 호환 |
  | Graha Maitri | 5 | 정서적 호환 |
  | Gana | 6 | 기질 호환 |
  | Bhakut | 7 | 사회적/재정적 호환 |
  | Nadi | 8 | 건강/유전 호환 |

#### D-2. Dasha 동기화
- Subject와 관계인의 현재 Dasha 기간 비교
- **동기화 패턴**:
  - 둘 다 확장기 (Jupiter/Venus Dasha) → 함께 성장
  - 한쪽 확장, 한쪽 수축 → 불균형, 한쪽이 끌어줘야 함
  - 둘 다 수축기 (Saturn/Rahu Dasha) → 함께 인내 필요

#### D-3. Transit 교차점
- 주요 행성 Transit이 두 사람의 차트에 동시에 영향을 미치는 시점
- 관계의 위기/기회 타이밍 예측

## Lens 3: 주역 (I Ching) — 실시간 의사결정 오라클

### A. 괘 도출 (Hexagram Casting)

#### 점괘 방법
- **동전법 (Coin Method)**: 동전 3개를 6회 던져 64괘 중 하나 도출
- **서죽법 (Yarrow Stalk Method)**: 50개 시초를 분류하여 괘 도출 (전통적, 더 정밀)
- **매화역수 (Plum Blossom Method)**: 시간/숫자 기반 자동 도출 (분석가가 계산)

#### 질문 설정 원칙
1. **구체적**: "내가 A를 해야 하는가?" (O) / "미래가 어떤가?" (X)
2. **현재 중심**: 현재 고민에 대한 지혜를 구함
3. **개방적**: 예/아니오가 아닌 방향성 질문
4. **진정성**: 진지한 마음가짐으로 질문

### B. 변효 해석 (Moving Lines)

| 요소 | 해석 |
|------|------|
| **본괘 (Primary Hexagram)** | 현재 상황의 본질 |
| **변효 (Moving Lines)** | 변화의 핵심 포인트, 가장 중요한 조언 |
| **지괘 (Transformed Hexagram)** | 상황이 전개될 방향 |
| **호괘 (Nuclear Hexagram)** | 숨겨진 내면의 역학 |

#### 효 위치별 의미 (6개 효)
| 위치 | 상징 | 의미 |
|------|------|------|
| 초효 (1) | 발 | 시작, 기초, 하층민 |
| 이효 (2) | 다리 | 내면의 중심, 관리자 |
| 삼효 (3) | 허리 | 전환점, 위기와 기회의 경계 |
| 사효 (4) | 가슴 | 외부 진입, 대신/측근 |
| 오효 (5) | 얼굴 | 지도자 위치, 최고 길위 |
| 상효 (6) | 머리 | 극한, 과잉, 물러남의 때 |

### C. 괘상 종합 (Hexagram Synthesis)

#### 팔괘 (8 Trigrams) 기본 속성
| 괘 | 이름 | 자연 | 오행 | 성질 |
|----|------|------|------|------|
| ☰ | 건(乾) | 하늘 | 金 | 강건, 창조 |
| ☷ | 곤(坤) | 땅 | 土 | 수용, 포용 |
| ☳ | 진(震) | 우레 | 木 | 움직임, 시작 |
| ☴ | 손(巽) | 바람 | 木 | 유순, 침투 |
| ☵ | 감(坎) | 물 | 水 | 위험, 지혜 |
| ☲ | 리(離) | 불 | 火 | 밝음, 아름다움 |
| ☶ | 간(艮) | 산 | 土 | 멈춤, 명상 |
| ☱ | 태(兌) | 연못 | 金 | 기쁨, 소통 |

#### 사주 오행과의 교차 분석
- 도출된 괘의 오행이 Subject의 용신/기신과 어떤 관계인지 확인
- 괘의 오행이 용신과 같으면 → 긍정적 방향, 행동 권고
- 괘의 오행이 기신과 같으면 → 신중한 접근, 대기 권고

### D. 실천 지침 (Action Guidance)

주역은 **실시간 의사결정 도구**로서 다른 3개 렌즈가 제시하는 장기적 흐름 위에 "지금 이 순간의 지혜"를 더함:

| 적용 영역 | 주역이 답하는 질문 |
|-----------|-------------------|
| 커리어 | "이 직장을 옮겨야 하는가?" "이 사업을 시작해야 하는가?" |
| 관계 | "이 사람과의 관계를 어떻게 대해야 하는가?" |
| 이사/이동 | "이 도시로 이사하는 것이 적절한가?" |
| 타이밍 | "지금이 행동할 때인가, 기다릴 때인가?" |
| 건강 | "현재 건강 관리에서 주의할 점은?" |

## Lens 4: 수비학 (Numerology) — 숫자 진동 분석

### A. Life Path Number (생명경로수)

생년월일을 환원하여 도출하는 가장 핵심적인 숫자. 인생의 목적과 방향을 나타냄.

#### 계산법
```
예: 1992년 12월 15일
1 + 9 + 9 + 2 = 21 → 2 + 1 = 3
1 + 2 = 3
1 + 5 = 6
Life Path = 3 + 3 + 6 = 12 → 1 + 2 = 3
```

#### Life Path 의미 요약
| 번호 | 키워드 | 특성 |
|------|--------|------|
| 1 | 리더, 독립 | 개척자, 자기 주도적, 독창적 |
